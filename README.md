# Anjali Ruprela — Data Analyst Portfolio

A premium, recruiter-focused portfolio built with Next.js, TypeScript, Tailwind CSS,
Framer Motion, and Recharts. Light theme, fully responsive, accessible, and SEO-ready.

## Tech stack
- **Next.js 14** (App Router) + **TypeScript**
- **Tailwind CSS** design system (blue `#2563EB` / teal `#14B8A6` / gold `#F59E0B`)
- **Framer Motion** — scroll reveals, counters, micro-interactions, typewriter hero
- **Recharts** — interactive revenue, segmentation, sales, and category charts
- **Lucide React** icons

## Sections
Hero · About · Skills · Projects (case studies) · Dashboard Showcase · Certifications ·
Education · Analytics Journey · Contact · Footer

## Getting started
```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # production build
```

## Personalize
- All copy, links, and project data live in `src/lib/data.ts`.
- Replace `public/Anjali-Ruprela-Resume.pdf` with the real resume.
- Update `linkedin`, `github`, `email`, and `calendly` URLs in `src/lib/data.ts`.

## Deploy
Push to GitHub and import into **Vercel** — zero config.
